# BookSwap

BookSwap is a web application for exchanging books between users.

## Business Idea

Users can add books for exchange. Featured books are highlighted and displayed first. 
The platform uses a free listing model with optional featured promotion managed by the admin.

## Features

- Add books
- Edit books
- Delete books
- Search books
- Featured books
- PostgreSQL database
- Docker support
- Responsive card layout

## Technologies

- ASP.NET Core MVC
- Entity Framework Core
- PostgreSQL
- Docker
- Bootstrap
- HTML/CSS

## Run Project

```bash
docker compose up --build
```

Open:

```text
http://localhost:8080
```
